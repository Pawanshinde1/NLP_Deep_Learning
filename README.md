Language Detection using NLP <br>
📌 Overview  <br>
This project detects the language of input text using Natural Language Processing and Machine Learning.<br>

🚀 Features<br>
Text preprocessing<br>
TF-IDF vectorization<br>
Naive Bayes & Logistic Regression models<br>
Model comparison<br>
Custom language prediction<br>
📊 Results<br>
Naive Bayes Accuracy: ~68%<br>
Logistic Regression Accuracy: ~73%<br>
🛠 Technologies Used<br>
Python<br>
Scikit-learn<br>
NLTK<br>
Pandas<br>
Matplotlib<br>
📁 Dataset<br>
Kaggle Language Detection Dataset<br>

▶️ How to Run<br>
Install dependencies<br>
Run notebook in Google Colab / Jupyter<br>
📌 Conclusion<br>
Logistic Regression performed better than Naive Bayes for this task.
