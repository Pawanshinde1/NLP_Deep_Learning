Language Detection using NLP <br><br>
📌 Overview  <br>
This project detects the language of input text using Natural Language Processing and Machine Learning.<br>

🚀 Features<br><br>
Text preprocessing<br>
TF-IDF vectorization<br>
Naive Bayes & Logistic Regression models<br>
Model comparison<br>
Custom language prediction<br><br>
📊 Results<br><br>
Naive Bayes Accuracy: ~68%<br>
Logistic Regression Accuracy: ~73%<br><br>
🛠 Technologies Used<br><br>
Python<br>
Scikit-learn<br>
NLTK<br>
Pandas<br>
Matplotlib<br><br>
📁 Dataset<br><br>
Kaggle Language Detection Dataset<br><br>

▶️ How to Run<br><br>
Install dependencies<br>
Run notebook in Google Colab / Jupyter<br><br>
📌 Conclusion<br>
Logistic Regression performed better than Naive Bayes for this task.
